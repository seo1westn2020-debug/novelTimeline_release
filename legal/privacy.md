## 1. 핵심 원칙 — "내 데이터는 내 기기에"

- 이용자가 작성한 **원고, 인물, 사건, 복선, 관계도, 수집 자료** 등 모든 프로젝트 데이터는 **이용자의 로컬 기기**(데스크톱의 파일 시스템 또는 브라우저의 IndexedDB)에만 저장됩니다.
- 제공자는 이러한 데이터에 **접근할 수 없고, 전송받지 않고, 분석하지 않습니다.**
- AI 기능(집필 보조·설정 검증 등)은 이용자가 직접 발급한 AI 제공자(Anthropic/OpenAI/Google)의 API 키를 사용해 **해당 제공자에게 직접 호출**됩니다. 제공자의 서버는 이 호출 경로에 포함되지 않습니다.

## 2. 수집되는 정보

제공자가 실제로 수집·처리하는 정보는 **결제 및 라이선스 검증 목적**에 한정됩니다.

### 2.1 Lemon Squeezy (결제 처리 위탁)
Lemon Squeezy가 결제 처리자(Merchant of Record)로서 다음 정보를 수집합니다. 제공자는 Lemon Squeezy 대시보드를 통해 필요한 범위에서만 열람합니다.

| 항목 | 목적 | 보관 기간 |
|---|---|---|
| 이름 | 영수증·세금 신고 | 결제 발생 후 **5년** (국세청 요건) |
| 이메일 주소 | 라이선스 키 전달, 환불·갱신 안내, 영수증 발송 | 라이선스 유효 기간 + 5년 |
| 결제 수단 정보 (카드 마스킹, 국가 등) | 부정 결제 방지 | Lemon Squeezy 정책에 따름 |
| 청구지 주소(국가 수준) | VAT·부가세 처리 | 5년 |

실제 결제 카드 번호 전문은 **제공자가 직접 수집·보관하지 않습니다** — Lemon Squeezy의 PCI DSS 준수 환경에서만 처리됩니다.

### 2.2 라이선스 검증 서버
라이선스 키의 유효성 검증 시 다음 정보가 제공자(또는 Lemon Squeezy License API)로 전송됩니다.

| 항목 | 목적 | 보관 기간 |
|---|---|---|
| 라이선스 키 | 유효성·만료 판정 | 라이선스 유효 기간 |
| 앱 버전 | 호환성 확인 | 로그성 보관, 90일 후 익명 집계로만 |
| 대략의 요청 시점 (timestamp) | 남용 감지 (단시간 과다 호출 방지) | 로그성, 90일 |

### 2.3 애플리케이션이 수집하지 않는 정보
다음은 **결코 수집되지 않습니다**:

- 이용자가 작성한 원고·메모·각주·설정 자료 본문
- 첨부한 이미지·파일의 내용
- IP 주소 (제공자 서버 기록에 남지 않음)
- 이용자의 기기 식별자(디바이스 ID, MAC 주소 등)
- 사용 패턴 분석(analytics)·행동 추적 쿠키·광고 식별자

## 5. 보안 조치

- Lemon Squeezy 대시보드 접근은 **2단계 인증(2FA)** 로 보호됩니다.
- 라이선스 API 통신은 **HTTPS(TLS)** 로 암호화됩니다.
- 이용자 원고는 로컬 저장이므로 **이용자의 기기 보안**(OS 로그인 암호, 전체 디스크 암호화 등)이 핵심 보호 수단입니다. 정기 백업을 권장합니다.
